# SuperSwingHelper

초보 부업 스윙 투자자를 위한 코스피 신호 알림 앱입니다.
이 앱은 자동매매가 아니라, 매수/매도 판단에 참고할 수 있는 신호를 쉽게 보여주는 용도입니다.

## 핵심 기능
- 워치리스트 관리
- 종목코드(6자리) 또는 종목명으로 종목 검색/추가
- 실시간 신호 대시보드
- 신호 유형: 매수 후보 / 돌파 감시 / 매도 경고
- 차트 탭에서 가격 흐름 + 주요 지표 확인
- 시간프레임 전환(1m, 5m, 15m, 1h)
- 신호 로그 확인 및 삭제(전체/종목별/개별)
- 포지션 수동 관리(미보유/보유중, 진입가, 수량, 손절/익절)
- 장중/장종료 상태 표시 및 상태에 맞는 갱신 동작
- 데이터 모드 전환: Mock 모드 / KIS 실데이터 모드

## 사용자 관점 사용 흐름
1. 워치리스트에 관심 종목 추가
2. 대시보드에서 신호 확인
3. 차트 탭에서 흐름과 신호 이유 확인
4. 필요 시 포지션 정보(진입가/수량 등) 업데이트

## 이 앱이 하지 않는 것
- 자동매매
- 주문 실행
- 수익 보장

## 보안/키 관리 안내
- KIS API Key/Secret은 로컬 환경에서만 유지하세요.
- 저장된 키는 Git에 커밋하지 않도록 기본 설정되어 있습니다.
- 이 저장소에는 키 원문이 올라가지 않게 관리합니다.

## 참고
- 이 앱의 신호는 투자 참고용입니다.
- 최종 매매 판단과 책임은 사용자 본인에게 있습니다.
